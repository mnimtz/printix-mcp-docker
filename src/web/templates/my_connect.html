Claude.ai
Plus, Pro, Team oder Enterprise
  1. Öffne claude.ai → Settings → Connectors.
  2. Klicke auf „Add custom connector".
  3. Name: z.B. Printix. URL einfügen:
    {{ mcp_url }}
  4. Klicke „Connect". Claude leitet auf den Login deines Printix-MCP-Servers weiter.
  5. Nach dem Login werden die Tools importiert. Test mit der Frage „Welche Printix-Tools stehen mir zur Verfügung?".
💡 Trust connector aktivieren, falls dir die Permission-Prompts pro Tool zu aufdringlich sind — Claude akzeptiert dann unsere ToolAnnotations und prompt nur noch bei wirklich destruktiven Aktionen.
💬
ChatGPT
Plus, Team oder Enterprise mit MCP-Konnektor-Support
  1. Öffne ChatGPT → Settings → Connectors (alternativ Custom GPTs).
  2. Wähle „Add new connector".
  3. Server-URL (SSE-Endpoint):
    {{ sse_url }}
  4. Authentifizierung: OAuth 2.0. Daten aus dem Verbindungsprofil oben einfügen (Client ID, Client Secret, Authorize-URL, Token-URL).
  5. Speichern. ChatGPT führt OAuth-Flow durch — danach sind die Tools verfügbar.
💻
Claude Code (Desktop CLI)
Anthropic's Code-Agent für macOS, Windows, Linux
  1. Im Terminal:
    claude mcp add --transport http printix {{ mcp_url }}
  2. Beim ersten Aufruf wird der Browser für den OAuth-Login geöffnet — anmelden mit deinem Printix-MCP-Account, fertig.
  3. Verifizieren mit:
    claude mcp list
